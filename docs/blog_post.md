# 유튜브에서 본 맛집·카페·여행지, 항상 "어디였지?" 하고 찾았던 당신께

안녕하세요. 작은 서비스 하나를 만들었습니다.

유튜브를 보다 보면 속초의 감성 카페, 부산의 숨은 맛집, 제주의 캠핑 명소 같은 **너무 가보고 싶은 장소**를 만나게 됩니다. 그럴 때마다 저장해놓고, 또 까먹고, 다시 찾고… 이 과정을 반복하다 보니 문득 이런 생각이 들었어요.

*"이 정보들을 한곳에 모아서 지도로 보여주면 얼마나 좋을까?"*

그렇게 **액시브(axiv)** 를 시작했습니다.

---

## 🤔 왜 만들었나?

사실 유튜브에는 엄청난 양의 장소 정보가 쌓여 있습니다. 맛집 리뷰, 카페 투어, 캠핑장 탐방, 여행 브이로그… 유튜버들이 직접 가보고, 먹어보고, 경험한 생생한 정보들 말이죠.

문제는 이 정보들이 **영상 속에 갇혀 있다**는 겁니다.

- "저 가게 이름이 뭐였지?" → 다시 영상 돌려보기
- "주소가 뭐였더라?" → 댓글 스크롤 내리기
- "전화번호는?" → 네이버에 다시 검색

이런 과정을 612번 반복하면서 느꼈습니다. *"이거 하나의 서비스로 만들면 되겠네."*

그렇게 유튜버가 추천한 장소들만 모아서 지도에 정리하는 **액시브(axiv)** 를 만들게 되었습니다.

## 🔍 액시브만의 특징

### 1. 유튜버 리뷰 기반 + 교차검증

가장 중요하게 생각한 것은 **정보의 정확성**입니다.

유튜브 영상에서 AI가 장소명을 추출하면, 실제 주소와 매칭되는지를 **네이버 검색 + Google Geocoding**으로 교차검증합니다. 신뢰도가 낮은 정보는 자동 필터링되며, 이상한 장소는 관리자가 직접 확인할 수 있는 시스템을 갖추고 있습니다.

### 2. 실질적인 정보 제공

단순히 "여기 좋아요"가 아닌, **실제 방문에 필요한 정보**를 제공합니다:

- **전화번호 93% 보유** (612곳 중 568곳)
- **영업시간 60% 보유**
- **메뉴/가격 정보 43% 보유**
- 웨이팅·주차 팁, 크리에이터 리뷰 텍스트

정보가 없는 장소는 지속적으로 보강하고 있습니다.

### 3. 유튜브 영상과 직접 연결

각 장소는 관련 유튜브 영상과 연결되어 있어, 해당 **타임코드**로 바로 이동할 수 있습니다. 유튜버가 어떤 말을 하면서 추천했는지 영상으로 바로 확인할 수 있어요.

## 🗺️ 사용 방법

사용법은 아주 간단합니다.

1. **axiv.vercel.app**에 접속
2. 지도에서 보고 싶은 지역으로 이동
3. 카테고리(맛집/카페/캠핑/낚시 등) 선택
4. **내 위치** 버튼 누르면 근처 장소 자동 검색

데스크톱과 모바일 모두에서 사용할 수 있고, 회원가입 없이도 모든 장소를 확인할 수 있습니다.

## 🛠️ 기술 이야기 (기술 덕후용)

액시브의 기술 스택을 간략히 소개하면:

| 영역 | 사용 기술 |
|---|---|
| **프론트엔드** | Next.js 16, React 19, TypeScript, Tailwind CSS |
| **지도** | Leaflet + react-leaflet (구글 맵 대신 무료!) |
| **상태 관리** | Zustand |
| **DB** | Supabase (PostgreSQL) |
| **AI 분석** | NVIDIA Nemotron API로 자막 분석 + 장소 추출 |
| **데이터 수집** | Python (yt-dlp + AI 교차검증) |

가장 흥미로웠던 부분은 **데이터 파이프라인**입니다.

유튜브 URL 하나만 넣으면:

```
① yt-dlp로 자막 다운로드
② AI가 장소명/주소/카테고리/리뷰 추출
③ 네이버 검색 + 지오코딩으로 정보 교차검증
④ Supabase DB에 저장
⑤ 전화번호·영업시간·메뉴 보강
```

이 모든 과정이 **자동화**되어 있습니다. 덕분에 꾸준히 장소 데이터를 추가할 수 있었고, 앞으로도 계속 업데이트할 예정입니다.
